A TTS GUI/wrapper for Piper-TTS
Made in Python (V 3.14.4 as of May 23 2026)

Example voice is from Keithito.com, and is Public Domain, and this project thanks them for making it public domain
Everything else is under GPL 3, see LICENSE


To use this project, Head to the releases section for information.


Inspired by a Blue Mute Cat.


Other voices can be used, https://rhasspy.github.io/piper-samples/ has the list and samples of everything, just download the .onnx file following the DOWNLOAD link on each voice sample, and update the voice.cfg file with the new voice file name, should be something like `model_path = ./en_US-VOICENAME.onnx`

Ensure the ./ is not removed, and everything after it is the filename (including the file extension .onnx)
